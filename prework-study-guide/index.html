Prework Study Guide

HTML

  • Hyptertext Markup Language
  • Gives content structure
  • Allows us to define elements like paragraphs, images, links, forms, and more

Elements

  • Work together to build structure
  • Provide framework that makes it easy for user navigation
  • Composed of opening and closing tags called angle brackets < >

Structure

Elements Preceding Head

  • !DOCTYPE : delcaration tells browser what document type it should expect when opened
  • html : opening tag which represents root/start of html document
  • lang= : tells the browser the language of the webpage

Head

  • Contains info or metadata that helps the browser render the page correctly
  • Elements included within the head are not visible to user
  • Attributes are defined at the beginning, not the end
  • Metadata contains important info about accessibility, search engines, and preformance
Metadata Tags < >
  • title : defines title for webpage, important for SEO
  • style : defines the CSS stles associated
  • base : defines the base URL
  • link : connects an external resource to the HTML doc
  • meta : defines metadata such as character set (charset), description, keywords, author, and viewport
  • script : embeds client-side scripts in the HTML doc

Body

  • Contains the content that is shown to the user
  • Elements that we want the user to see and interact with must be placed inside this element
  • Body = parent element
  • Anything within the body = child element
  • Content contained between opening and closing tags
  • Closing tags begin with a forward slash
Body Tags < >
  • h# : heading, page organization (h1, h2, h3, h4, h5, h6)
  • p : paragraph
  • ul : unordered list (bullets)
  • ol : ordered list (numbers)
  • li : list items
  • img : info about images
  • src : img attribute, defines location/source of image
  • alt : text string that describes appearance and functionality of image
  • br : line break, creates line of empty space between content
  • a : anchor, creates link to webpage or other webpages
  • href : points to the URL for the link
  • footer : appears at bottom of page, info about author, copyright, contact, sitemap, and navigation
